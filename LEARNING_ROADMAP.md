## üí° Study Tips

### Daily Routine
1. **Review** (10 min): Review yesterday's concepts
2. **Learn** (20 min): Read about today's topic
3. **Practice** (40 min): Complete 1-2 tasks
4. **Test** (10 min): Run tests and debug
5. **Reflect** (10 min): Write notes about what you learned

### When Stuck
1. Re-read the function docstring carefully
2. Write out the algorithm in pseudocode
3. Break the problem into smaller steps
4. Test with simple examples by hand
5. Only then look at hints or solutions

### Testing Strategy
```bash
# Test as you go
python -m pytest tests/test_algorithms.py::TestSorting::test_bubble_sort -v

# See detailed output
python -m pytest tests/ -v --tb=short

# Run only failed tests
python -m pytest tests/ --lf
```
